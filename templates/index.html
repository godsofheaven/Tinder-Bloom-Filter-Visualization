How Tinder Uses Bloom Filters

Problem: Tinder needs to quickly check if you've already swiped on a profile without storing every single swipe in memory.

Solution: Use a Bloom filter to track swiped profiles. It can tell you:

  • "Definitely haven't swiped" - Show this profile
  • "Might have swiped" - Double-check database (rare)
